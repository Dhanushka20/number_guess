# Number Guessing Game

Welcome to the Number Guessing Game! This is a simple Python game where players have to guess a random number generated by the computer within a specified range.

## Introduction

Number Guessing Game is an interactive project that challenges players to guess a random number. With simple rules and intuitive gameplay, players can test their guessing skills and see how many attempts it takes them to guess the correct number.

## Features

- Random number generation within a specified range.
- Interactive user input for guessing the number.
- Feedback messages to guide players based on their guesses.
- Win condition when the player guesses the correct number.

## How to Play

1. **Enter the Range**: Type a number to set the upper limit of the range for the random number generation.
2. **Guess the Number**: Enter your guess when prompted.
3. **Feedback**: Receive feedback on whether your guess is higher or lower than the actual number.
4. **Win Condition**: Keep guessing until you guess the correct number.
5. **Game Over**: After guessing the correct number, see how many attempts it took you to win.

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/number-guessing-game.git
   ```

2. Navigate to the project directory:

   ```bash
   cd number-guessing-game
   ```

3. Run the game:

   ```bash
   python number_guessing_game.py
   ```
